# Build a Hooper

A simple static basketball player builder powered by 2025-26 NBA season stat benchmarks.

Users can tune position, measurements, skill priorities, and physical tools. The app returns a projected stat line and matches the build to a 2025-26 statistical lane.

## GitHub Pages

This site is built with plain HTML, CSS, and JavaScript, so it can be hosted directly from the repository root with GitHub Pages.

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to Settings > Pages.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and `/root`.
